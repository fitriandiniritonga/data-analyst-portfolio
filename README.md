## Featured Projects
### 1. Hybrid ARIMA–Metaheuristic Modeling for High-Volatility Time Series Forecasting  
**Adaptive Parameter Tuning with GA, PSO, and Continuous Hybrid Optimization**  
Developing an advanced time series forecasting framework by integrating ARIMA with Genetic Algorithm (GA), Particle Swarm Optimization (PSO), and hybrid optimization strategies to improve predictive performance under volatile, non-stationary, and heteroskedastic conditions. This project focuses on adaptive parameter initialization, hybrid GA→PSO and PSO→GA comparisons, volatility-aware fitness formulation, and continuous optimization mechanisms to enhance forecasting robustness beyond conventional ARIMA approaches.
