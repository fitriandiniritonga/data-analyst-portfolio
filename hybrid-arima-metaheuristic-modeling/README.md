# Hybrid ARIMA–Metaheuristic Modeling for High-Volatility Time Series Forecasting

## Project Overview
This project develops an advanced forecasting framework by integrating ARIMA with Genetic Algorithm (GA), Particle Swarm Optimization (PSO), and hybrid optimization strategies to improve predictive accuracy under volatile, non-stationary, and heteroskedastic time series conditions.

---

## Research Objectives
- Improve ARIMA forecasting performance through metaheuristic optimization
- Compare GA, PSO, GA→PSO, and PSO→GA strategies
- Develop adaptive parameter tuning mechanisms
- Design volatility-aware and multi-objective fitness functions
- Enhance forecasting robustness for complex time series
